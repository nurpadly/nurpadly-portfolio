```
nurpadly-portfolio/
├── index.html              ← Homepage
├── projects.html           ← Case Study / Project Detail
├── admin.html              ← Admin Gallery (Cloudinary)
├── admin-projects.html     ← Admin Projects CRUD
├── assets/
│   ├── gallery.json        ← Database karya gallery ⭐
│   ├── projects.json       ← Database project detail ⭐
│   └── gallery/            ← (kosong, gambar di Cloudinary)

```

---

## Cara Update Gallery (admin.html)

1. Buka `admin.html` → login password `nurpadly2024`
2. Upload gambar → otomatis ke Cloudinary
3. Klik **"Download gallery.json"**
4. Replace file `assets/gallery.json` dengan yang baru
5. Upload ulang folder ke Netlify → selesai!

---

## Cara Update Projects (admin-projects.html)

1. Buka `admin-projects.html` → login password `nurpadly2024`
2. Tambah/edit/hapus proyek
3. Klik **"Download projects.json"**
4. Replace file `assets/projects.json` dengan yang baru
5. Upload ulang folder ke Netlify → selesai!
